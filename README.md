# AssistEye

Мобильное приложение (Flutter) для помощи слабовидящим и незрячим пользователям. Голосовое управление, распознавание текста (OCR), детекция объектов и определение купюр — **всё работает офлайн**, без передачи данных на сервер.

## Возможности (MVP)

| Команда | Описание |
|---------|----------|
| **«Прочитай»** | OCR — распознавание печатного текста (RU/EN) |
| **«Опиши» / «Что это»** | Детекция бытовых объектов (SSD MobileNet v2) |
| **«Купюра»** | Определение банкнот ₽ и $ *(feature flag)* |
| **«Помощь»** | Список доступных команд |
| **«Повтори»** | Повторное озвучивание последнего результата |
| **«Стоп»** | Остановка озвучивания |

## Технологический стек

- **Framework:** Flutter (Dart) ≥ 3.x
- **ASR:** Vosk (офлайн, RU + EN)
- **OCR:** Google ML Kit Text Recognition v2 (fallback: Tesseract)
- **CV:** TensorFlow Lite — SSD MobileNet v2 INT8
- **TTS:** Системные голоса (flutter_tts)
- **State management:** flutter_bloc
- **Storage:** SQLite (sqflite) + flutter_secure_storage
- **ML-скрипты:** Python 3.10+, TensorFlow
