# Assist Eye

**Assist Eye** — это мобильное приложение для ассистирования, построенное на базе React Native (Expo) и интегрирующее алгоритмы компьютерного зрения. Проект сочетает в себе кроссплатформенный клиент и исследовательскую базу в виде Jupyter-ноутбуков для разработки и анализа моделей машинного обучения.

## Стек технологий

*   **Мобильная разработка:** React Native, Expo, TypeScript, JavaScript
*   **Исследования и ML/CV:** Python, Jupyter Notebooks (разработка моделей обработки изображений и компьютерного зрения)
*   **Тестирование:** Jest
*   **Качество кода:** ESLint, Prettier

---

## Структура проекта

Репозиторий имеет следующую структуру директорий:

*   **`notebooks/`** — исследовательская часть проекта. Содержит Jupyter-ноутбуки, в которых проводятся эксперименты по компьютерному зрению (CV) и машинному обучению (ML) (например, детекция лиц, разметка данных или трекинг направления взгляда).
*   **`src/`** — основной исходный код мобильного приложения.
*   **`components/`** — переиспользуемые UI-компоненты приложения (элементы интерфейса, кастомные кнопки, карточки).
*   **`constants/`** — глобальные константы проекта (цветовая палитра, конфигурационные параметры, шрифты).
*   **`context/`** — контексты React для централизованного управления состоянием приложения (например, управление потоком камеры или сессиями калибровки).
*   **`plugins/`** — конфигурационные плагины Expo для тонкой настройки нативных модулей.
*   **`patches/`** — исправления (патчи) сторонних npm-зависимостей.
*   **`android/`** — нативные файлы сборки под платформу Android (генерируются Expo в процессе prebuild).

## Быстрый старт

Для запуска мобильного приложения на локальной машине или мобильном устройстве выполните следующие шаги:

### Требования
*   Установленная среда [Node.js](https://nodejs.org/) (рекомендуется LTS-версия).
*   Инструменты [Expo CLI](https://docs.expo.dev/get-started/installation/).
*   Для работы с ноутбуками: установленный Python и среда Jupyter (JupyterLab или расширение VS Code).

### 1. Клонирование репозитория
```bash
git clone https://github.com/H1merka/assist-eye.git
cd assist-eye
```

### 2. Установка зависимостей
Установите необходимые пакеты для мобильного приложения:
```bash
npm install
# или с использованием yarn
yarn install
```

### 3. Настройка окружения
Создайте файл .env на основе примера и укажите необходимые конфигурационные параметры:
```bash
cp .env.example .env
```

### 4. Запуск приложения в режиме разработки
Запустите сервер разработки Expo:
```bash
npx expo start
# или
npm start
```
После запуска вы можете отсканировать QR-код с помощью приложения Expo Go (на Android) или запустить проект в симуляторе.

## Тестирование и качество кода
В проекте настроены инструменты для обеспечения качества кода и юнит-тестирования:

Запуск тестов Jest:
```bash
npm run test
```
Проверка кода линтером (ESLint):
```bash
npm run lint
```
